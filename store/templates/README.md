# 🛒 Simple E-Commerce Store

A simple and responsive E-Commerce Web Application built using Django Framework.  
This project allows users to browse products, manage cart items, add wishlist products, and place orders.

---

## 🚀 Features

- 🏠 User-friendly Home Page
- 🔍 Product Search Functionality
- 📱 Product Details Page
- 🛒 Add to Cart System
- ❤️ Wishlist Management
- ❌ Remove Wishlist Items
- ⚡ Buy Now Option
- 📦 Order History
- 👤 User Profile Page
- 🔐 User Registration
- 🔑 Login / Logout System
- ⭐ Product Rating Display
- 🎨 Responsive UI Design

---

## 🛠️ Technologies Used

- Python
- Django Framework
- HTML5
- CSS3
- SQLite Database
